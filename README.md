# Sistema de Pesquisa de Satisfação - Volkswagen Capivari 🚗

Este projeto foi desenvolvido como parte do **Projeto Integrador** para o curso de **Engenharia de Computação da UNIVESP**. O objetivo é modernizar a coleta de feedbacks de clientes em uma concessionária automotiva.

## 🚀 Funcionalidades Implementadas
- **Interface Responsiva (Mobile First):** Layout adaptado para smartphones e desktops usando HTML5 e CSS3.
- **Máscaras Inteligentes (JS):** Formatação automática de datas e placas (padrão antigo e Mercosul) em tempo real.
- **Backend Robusto:** Desenvolvido em **Python (Flask)** com persistência de dados em **SQLite** via SQLAlchemy.
- **Automação de Alertas (WhatsApp API):** Integração com o CallMeBot para notificar a gerência instantaneamente em casos de avaliações negativas.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Framework Web:** Flask
- **Banco de Dados:** SQLite (SQLAlchemy)
- **Frontend:** HTML5, CSS3 (Card UI Design), JavaScript Vanilla
- **Integração:** API CallMeBot (WhatsApp)

## 📋 Como rodar o projeto localmente
1. Clone o repositório:
   `git clone https://github.com/Edk1990/Projeto_PI.git`
2. Instale as dependências:
   `pip install flask flask-sqlalchemy requests`
3. Inicie o servidor:
   `python app.py`
4. Acesse no navegador:
   `http://127.0.0.1:5000/tela_login.html`
